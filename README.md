# 📊 Bank Loan Analysis Dashboard

## 📌 Project Overview

The **Bank Loan Analysis Dashboard** is an interactive data visualization project designed to analyze loan applications, funding performance, and borrower risk segmentation.

It provides insights into:

* Loan distribution (good vs bad loans)
* Funding and repayment trends
* Borrower characteristics (employment length, purpose, home ownership)
* Key financial indicators such as interest rates and debt-to-income ratio (DTI)

The dashboard is divided into three main sections:

1. **Summary**
2. **Overview**
3. **Details** *(not shown here but part of the complete project)*

---

## 📊 Key Metrics (KPIs)

The dashboard highlights the following core performance indicators:

* **Total Loan Applications:** 38.6K
* **Total Funded Amount:** $435.8M
* **Total Amount Received:** $473.1M
* **Average Interest Rate:** 12.0%
* **Average DTI:** 13.3%

Each KPI includes:

* **MTD (Month-to-Date)** performance
* **MoM (Month-over-Month)** growth trends

## 📈 Overview Dashboard

The **Overview** page focuses on trends and borrower segmentation.

### 🔹 Monthly Trends

* Loan applications show steady growth throughout the year
* Peak observed in **December (~4.3K applications)**

### 🔹 Loan Term Distribution

* **36 months:** ~73.2%
* **60 months:** ~26.8%

Shorter-term loans dominate the portfolio.

### 🔹 Borrower Profile Analysis

#### 👔 Employment Length

* Highest applications from borrowers with **10+ years employment (8.9K)**
* Indicates stronger participation from experienced workforce

#### 💳 Loan Purpose

Top purposes include:

* Debt consolidation (largest segment)
* Credit card refinancing
* Home improvement

#### 🏠 Home Ownership

* Majority of applicants are **Renters**
* Followed by **Mortgage holders**

## ⚙️ Filters & Interactivity

Users can dynamically filter data using:

* **State**
* **Grade**
* **Loan Purpose**
* **Loan Quality (Good vs Bad)**
* **Selected Measures (in Overview page)**

These filters enable deep-dive analysis across different borrower segments.

---

## 📊 Insights & Observations

* High proportion of **good loans (86%)** indicates strong portfolio health
* **Debt consolidation** is the primary driver of loan demand
* Increasing **monthly application trend** suggests growing credit demand
* Borrowers with longer employment history are more active
* Renters form the largest borrower group
